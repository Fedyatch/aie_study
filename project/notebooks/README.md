# Ноутбуки проекта

В этой директории размещаются экспериментальные ноутбуки (Jupyter Notebook) для исследования данных, прототипирования моделей и проведения контролируемых экспериментов.

> **Важно**: Ноутбуки предназначены для **исследовательской работы**. Код, который становится частью продакшен-пайплайна (загрузка данных, предобработка, инференс), выносится в модули `src/` для обеспечения переиспользуемости, тестируемости и воспроизводимости.

---

## Доступные ноутбуки

| Файл | Описание | Статус |
|------|----------|--------|
| [`01_eda_baseline.ipynb`](01_eda_baseline.ipynb) | **Разведочный анализ данных (EDA)** для датасета *Give Me Some Credit*. Включает загрузку данных, анализ распределений, пропусков, выбросов, корреляционный анализ, baseline (`DummyClassifier`) и формулировку гипотез для предобработки. | Завершён |
| [`02_model_tuning.ipynb`](02_model_tuning.ipynb) | **Обучение и сравнение моделей**. Реализует полный пайплайн: предобработка (импутация, логарифмирование, агрегация просрочек, `RobustScaler`) → обучение `LogisticRegression`, `RandomForest`, `HistGradientBoosting` с подбором гиперпараметров (`GridSearchCV`) → оценка метрик (ROC-AUC, F1, Accuracy) → сохранение финального пайплайна и метаданных. | ✅ Завершён |

## Как запустить ноутбуки

### Требования
- Активированное виртуальное окружение (`uv` или `venv`)
- Установленные зависимости: `uv sync`
- Датасет `cs-training.csv` в директории `data/raw/`

### Запуск через Jupyter Notebook
```bash
cd project
uv run jupyter notebook notebooks/
