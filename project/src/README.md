# Исходный код сервиса кредитного скоринга

В этой директории находится production‑ready код REST API для оценки кредитного риска. Код организован в виде Python‑пакета `credit_scoring`, следует модульной структуре и использует статическую типизацию Pydantic. Архитектура разделяет зоны ответственности: загрузку модели, валидацию запросов, препроцессинг и HTTP‑интерфейс.

## Структура

- `api.py` – точка входа FastAPI. Управляет жизненным циклом приложения, регистрирует эндпоинты `/health` и `/predict`, логирует запросы и замеряет время обработки.
- `model.py` – класс `ScoringModel`. Загружает сериализованный пайплайн (`scorer_pipeline_v1.0.joblib`), содержит функции предобработки (`log1p_transform`, `add_total_past_due`), выполняет преобразование признаков и вызов `predict_proba`.
- `schemas.py` – Pydantic‑модели `CreditApplication` и `CreditResponse` для строгой валидации входящих JSON‑запросов и структурирования ответов.
- `__init__.py` – инициализация пакета.

## Запуск сервиса

cd project
