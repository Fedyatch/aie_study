# Тесты проекта

В этой директории находятся автоматизированные тесты для проверки корректности работы сервиса кредитного скоринга.

## Структура тестов

```text
tests/
├── init.py # Инициализация пакета тестов
├── conftest.py # Настройка путей для pytest
├── README.md # Этот файл
└── test_api.py # Интеграционные тесты FastAPI-сервиса
```

## Описание тестовых модулей

### `test_api.py`
Интеграционные тесты для `src/credit_scoring/api.py` (FastAPI):

| Тест | Описание |
|------|----------|
| `test_health` | Проверка эндпоинта `/health`: статус-код 200, наличие ключа `status` |
| `test_predict_valid` | Успешный запрос с валидной анкетой, проверка структуры ответа (`probability_default`, `risk_category`) |
